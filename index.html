How splitting works here: this prototype uses mid/side processing (vocal ≈ center channel, instrumental ≈ sides) — a real classic isolation trick that works best on stereo mixes with a centered lead vocal. It's a DSP approximation, not a trained source-separation model, so results vary by track.
▸ Harmony Mixer — Lead / Soprano / Alto / Tenor / Backing
How harmony parts work: each part is generated by granular-resynthesis pitch-shifting of your actual isolated vocal performance (same phrasing & timing, shifted by the chosen interval) — not a generic synth voice. The tone knob applies a live EQ character on top. This is a DSP approximation of voice-conversion harmony tools, built to run entirely in your browser.
▸ Cloud AI Voice Engine — Optional, requires your own server
This calls a DiffSinger score-to-singing model you deploy yourself (see DEPLOY.md). It's a different input than the rest of this app: instead of transforming your recorded vocal, you give it lyrics + note names + durations and it sings a brand new performance for that part. Leave the URL blank to keep using the local pitch-shift engine above.
